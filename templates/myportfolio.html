Photofy

Photofy is a photo album app that allows users to upload photos to their account. The user has the option to select if the photo is "private" or "public". All photos that are chosen to have a "public" setting are then shared with other users on the app. Photofy was developed using Django's framework and is currently hosted via Railway. The static files and images uploaded are hosted via AWS S3 Bucket.
